# MacroBalance Storefront (React)

A small end-to-end React storefront based on the provided site map:

- Home
- Shop All (grid)
- New Launches / New Arrivals
- About / Made With Purpose
- Login (account sections)
- Search (popular searches + suggestions)
- Cart
- Find Your Balance (quiz)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
